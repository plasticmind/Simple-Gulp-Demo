A Simple Gulp Demo

Project Description

This is a bare-bones boilerplate of Gulp. It only handles the basics:

  • Pre-compiles Sass
  • Watches for style changes
  • Live reloads changes via BrowserSync

You can do a lot more with Gulp, like deleting, renaming, or including files, auto-prefixing for backward compatibility, spinning up a PHP server, optimizing images, and even complex operations like extracting and inlining critical path CSS.

Requirements

Before you can run Gulp, you will need to install a couple of things:

  • Install Node.js
  • Install Gulp CLI by running npm install --global gulp-cli.

Getting Started

  • Download or clone the files from the repository on Github.
  • At the command line, navigate to the folder created by Git and run npm install to download the project's dependencies.
  • Run gulp. This starts the watch process and fires up a server with your main index file.
  • The page should come up in your browser automatically, but if it doesn't, the local server should be serving at http://localhost:8080/.
